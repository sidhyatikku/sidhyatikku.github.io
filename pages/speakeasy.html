speakEasy — Text-to-Speech Generator
This project was my final for Core 2: Interaction Studio class during Spring 2022 semester. The prompt was to create a website interface that takes an input from the user and give out an output. It could be an image, audio, video, sound, or text. For this prompt, I wanted to work on something that makes any text, book, article, an audiobook.
Execution
The project was uses text-to-speech synthesis JS. The website has a clean, minimal, and easy to use interface. The user has options to set the language, gender, tone, and pitch of the voice. The user can easily type or paste long texts and listen to the whole text.
