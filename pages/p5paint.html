p5.js Random Painting
This project was made for my Core 2: Interaction Studio class during Spring 2022. The prompt was to create something artistic with p5.js
Ideation
As a kid, I was into messing around with creative tools and one of my favourite tool was Tux Paint. The idea was to create a set of painting tools inspired by tux paint that changes size, colour, and has an entropy element which changes with cursor position.
Execution
This was my first time working with p5.js and I wanted to create something that had a human touch and at the same time looked generated by code, or maybe ask the question "What would a modern day Jackson Pollock painting look like?" The painting tools have a heavy focus on entropy and random colour scheme.
